# Python Sayı Bulucu

Bu proje, Python ile belirli bir aralıktaki kuvvetli sayıları bulmak için hazırlanmış basit bir algoritma çalışmasıdır.

## Projenin Amacı

Bu projede dosya okuma, dosyaya yazma, fonksiyon kullanımı ve sayı kontrol algoritmaları üzerine pratik yapılmıştır.

## Kullanım

1. `kuvvetlisayilar.txt` dosyasının içine kontrol edilecek aralık yazılır.

Örnek:

1,1000

2. Python dosyası çalıştırılır.

python sayi_bulucu.py

3. Bulunan sonuçlar `kuvvetlisayilar.txt` dosyasına yazılır.

## Dosyalar

- `sayi_bulucu.py` : Ana Python kodu
- `kuvvetlisayilar.txt` : Alt ve üst sınır bilgisinin bulunduğu dosya

## Not

Bu proje başlangıç seviyesi Python algoritma pratiği için hazırlanmıştır.
